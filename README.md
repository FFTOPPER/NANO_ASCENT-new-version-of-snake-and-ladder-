## Overview

NANO ASCENT is a modern browser-based 3D reinterpretation of the traditional Snake and Ladder game.  
Instead of a flat board, the player progresses vertically through a futuristic environment filled with platforms, ladders, snakes, power-ups, and animated camera movement.

The project combines nostalgia with modern web technologies to create a visually engaging gameplay experience.

---

## Features

- Fully playable in browser  
- 3D environment using Three.js  
- Smooth movement and animations  
- Dice roll mechanics  
- Ladder progression system  
- Snake hazard system  
- Shield power-up feature  
- Boost tiles for faster progress  
- Dynamic camera tracking  
- Responsive design for desktop and mobile  
- Public hosting using GitHub Pages

## How to Play

1. Click **ROLL DICE**
2. Move forward according to the dice result
3. Use ladders to climb faster
4. Avoid snakes that reduce progress
5. Collect shields for protection
6. Reach Tile 100 to win the game
